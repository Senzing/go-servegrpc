# servegrpc errors

## Error prefixes

Error identifiers are in the format `senzing-PPPPnnnn` where:

`P` is a prefix used to identify the package.
`n` is a location within the package.

Prefixes:

1. `6011` - g2config
1. `6012` - g2configmgr
1. `6013` - g2diagnostic
1. `6014` - g2engine
1. `6015` - g2hasher
1. `6016` - g2product
1. `6017` - g2ssadm## Errors
